This pack fixes the simple setup ↔ dashboard loop.

It removes the easy manual bounce:
- setup page no longer has "Skip to Dashboard"
- dashboard page no longer links straight back to setup

Install:
1. Unzip
2. Merge files into your project
3. Replace older files
4. Restart `npm run dev`

Use:
- home page for routing
- setup page for onboarding
- dashboard page after sync
